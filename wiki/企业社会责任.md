# 企业社会责任

[[中欧国际工商学院]] EMBA [[开学模块]]课程，由**芮萌教授**主讲（2025-10-29）。主题为"商业向善，新商业文明的底色"，覆盖个人修身、企业社会责任演进、双碳战略与人生使命四个层次。

## 核心命题

> 商业向善不是成本，而是通过赋能各利益相关方最终提升企业盈利能力与估值倍数的长期战略投资。"做难而正确的事"需要用长期主义视角看待股东价值。

## 一、第一性原理：认知的底层逻辑

课程从亚里士多德的**第一性原理**（First Principles）切入：每个系统都存在最基本的命题或假设，不可省略、不可违背。

### 各领域的第一性原理 vs 常见误区

| 领域 | 第一性原理 | 常见误区 |
|:---|:---|:---|
| 教育 | 培养能力 | 追求分数 |
| 健康 | 自律（管住口、迈开腿） | 出问题后盲目依赖医院 |
| 创业 | 满足市场真实需求 | 满足个人喜好 |
| 沟通 | 加深理解 | 说服对方 |
| 经济 | 系统性发展与福祉 | 唯GDP论 |
| **商业** | **推动社会经济进步** | **利润最大化** |

## 二、修身济天下：企业家的个人修为

### 好人做坏事的四大原因

1. **服从权威**：作为"打工人"受上方驱使
2. **满足预期**：迎合组织角色规范
3. **受到激励**：绩效制度的错误导向
4. **从众心理**：违规成本低，效仿他人

> 仅靠个人品德在商业环境中不够，必须将道德底线内嵌于组织治理结构中。

### 修身四要素

| 要素 | 内涵 | 典型案例 |
|:---|:---|:---|
| 底线 | 坚守道德底线，德不配位必有祸殃 | 中欧学员道德基准分1.53（极低容忍度） |
| 认真（工匠精神） | 工作注入生命和灵魂 | 日本25,000+百年老店；《寿司之神》小野二郎 |
| 契约精神 | 超越法律契约的"精神契约" | 纽约将军墓旁小墓碑200年契约传承 |
| 善良 | 聪明是天赋，善良是选择 | 贝索斯祖父教导：善良比聪明更难也更重要 |

## 三、CSR演进：从Feel Good到Doing Well by Doing Good

### 理论发展脉络

| 年份 | 里程碑 | 核心观点 |
|:---|:---|:---|
| 1969 | 弗里德曼股东理论 | 创造利润就是企业唯一的社会责任 |
| 1979 | 卡罗尔金字塔 | 经济→法律→伦理→慈善（四层递进） |
| 1984 | 利益相关者理论 | 企业需平衡顾客、员工、供应商等命运共同体 |
| 1997 | 三重底线 | 经济×环境×社会的交集才是可持续战略 |
| 2006 | ESG与B Corp | 联合国PRI：非财务性评价体系；共益企业认证 |
| 2011 | 创造共享价值（CSV） | 波特：初创阶段就应兼具商业与社会目标 |
| 2018 | 诺奖可持续发展理论 | 批评GDP外部性，要求环境与经济双轮驱动 |

### CSR三阶段

1. **Feel Good**（补偿心理）：赚钱后捐赠，弥补负罪感
2. **Looks Good**（品牌公关）：CSR作为营销工具
3. **Doing Well by Doing Good**（义利并举）：CSR融入核心商业战略

### 利益相关方回报机制

- **客户**：善因营销提供"情绪价值"→品牌忠诚度↑、复购率↑、价格脱敏、免费社交传播
- **员工**：公益培养跨界技能→流失率↓、敬业度↑、人效↑
- **环境**：从Recycle到Redesign/Reuse/Remanufacture→运营成本↓
- **社区/政府**：融入当地社区→政府白名单、政策参与权、牌照等稀缺资源
- **股东**：`股价 = PE × EPS`，ESG投入长期拉高估值倍数与每股盈利

## 四、双碳战略：中国的战略主动

### 为什么中国拥抱"3060双碳"

这**不是**屈服于西方压力，而是基于三重内生需求：

1. **经济质量转型**：中国碳排放强度6.9万吨/亿美元，是全球平均的1.8倍（GDP占17%，碳排放占28%）
2. **能源安全**：原油对外依存度73%，80%依赖马六甲海峡。新能源是摆脱受制的根本解法
3. **产业链跃升**：在新能源领域占据"微笑曲线"高附加值区间（风电装机占全球40%，光伏44%，水电29%）

### 破解胡焕庸线

西部拥有丰富的风光绿电资源。双碳战略下，廉价能源将吸引产业链西移，有望彻底改变东西部经济不平衡——这是1935年以来的历史性机遇。

## 五、新商业文明与人生第二座山

### 企业属性光谱

企业并非"纯商业"或"纯慈善"的二元对立，而是一个连续光谱：

| 类型 | 特征 | ROE |
|:---|:---|:---|
| 非盈利组织 | 纯慈善，无造血能力 | 0% |
| 社会企业 | 公益为主，市场为辅 | 0-5% |
| **共益企业（B-Corp）** | 综合价值共创 | 3-8% |
| 纯商业企业 | 完全逐利 | >8% |

### 儒家文化与新商业文明

西方达尔文"丛林法则"强调竞争淘汰；儒家的"仁义礼智"与"天地人和谐"为新商业文明提供了更高维的包容智慧。KK之问"Can China be cool?"的答案在于产品卓越、文化输出与城市魅力。

### 人生的第二座山

EMBA学员平均年龄41岁，正处于幸福感U型曲线谷底：

- **第一座山**：获取、构建自我、追求个人成就与物质
- **第二座山**：给予、摆脱自我、与社会并肩同行

> 墨西哥渔夫的寓言：哈佛MBA花一辈子折腾，最终追求的状态不过是渔夫每天已经拥有的——但真正的答案在于，渔夫从未经历过攀登与超越，而企业家可以选择用商业改变世界。
