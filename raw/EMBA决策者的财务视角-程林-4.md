# D. 详细记录

### 1. 业务场景：非现金赊购与现金费用支付
* **知识点**：资产负债表两边同增/同减的平衡关系。
* **一句话总结**：不论借贷，只看交易导致企业资产和负债/权益的具体增减变化。
* **详细示例与证据**：
  * **交易1（赊购存货1300）**：未付现金，存货明显增加1300，因此记【存货增加1300】；为了保持等式平衡，由于欠款，记【应付账款增加1300】。
  * **交易2（支付工资730）**：现金流出，记【现金减少730】；对应费用的增加会减少利润，即减少权益，记【权益（费用）减少730】。
* **❌ 常见误区**：认为非现金交易不需要立即记账。
* **✅ 纠正**：在权责发生制下，只要经济义务发生，即便没有现金流转也必须立即确认资产或负债的增减。

### 2. 业务场景：销售收入确认（现金与赊销）
* **知识点**：权责发生制下的收入确认原则。
* **一句话总结**：无论是否收到现金，只要货物出门或业务完成，就必须确认收入。
* **详细示例与证据**：
  * **交易3（现金销售收到1940）**：记【现金增加1940】，【权益（收入）增加1940】。
  * **交易4（赊销商品收入1810）**：未收到现金，但业务已完成。记【应收账款增加1810】，【权益（收入）增加1810】。
* **❌ 常见误区**：只有收到客户打款才算是企业的收入。
* **✅ 纠正**：权责发生制不等于现金制。货出门了，即使是赊销，也会增加“应收账款”并同步增加“所有者权益（收入）”。

### 3. 业务场景：成本结转
* **知识点**：存货减少与销售成本的对应关系。
* **一句话总结**：商品售出后，减少存货并等额记录成本（减少权益）。
* **详细示例与证据**：
  * **交易5（商品销售成本合计1280）**：直接使用总成本数据。记【存货减少1280】，计入成本即记【权益（成本）减少1280】。
* **❌ 常见误区**：试图搞清楚每笔总成本分别对应具体哪一笔明细销售订单，导致思维卡壳。
* **✅ 纠正**：在编制总账和报表时，只需关注总成本变化及资产负债表的整体变化，无需钻牛角尖追溯单笔明细。

### 4. 业务场景：收回应收与支付应付
* **知识点**：债权债务的结算与消除。
* **一句话总结**：收回应收和支付应付是资产内部的转换或资产与负债的同减，不影响当期收入和费用。
* **详细示例与证据**：
  * **交易7（收到应收账款1510）**：记【现金增加1510】，同时消除债权记【应收账款减少1510】。
  * **交易8（支付应付账款1720）**：记【现金减少1720】，消除债务记【应付账款减少1720】。
* **❌ 常见误区**：收到应收账款时再次确认收入，支付应付账款时再次确认费用。
* **✅ 纠正**：前面在赊销和赊购时已经确认过收入和费用，结清款项时绝对不能重复记账，否则会导致利润虚增或虚减。

### 5. 业务场景：预收款项处理
* **知识点**：预收账款属于企业负债而非收入。
* **一句话总结**：收到客户预付款但未发货时，不能确认收入，必须记为预收账款（负债）。
* **详细示例与证据**：
  * **交易9（收到预付款650）**：拿到现金但义务未完成。记【现金增加650】，增加负债记【预收账款增加650】。
  * **后续推演（假设发货350）**：义务完成时，记【预收账款减少350】，同时记【权益（收入）增加350】，再同步结转对应成本。

### 6. 业务场景：折旧与其他资产变动
* **知识点**：直线法折旧及资产负债表科目转换。
* **一句话总结**：固定资产折旧是资产价值的消耗，应作为期间费用处理。
* **详细示例与证据**：
  * **交易10/11（银行借款200及买资产800）**：借款记【现金+200，银行借款+200】；买资产记【现金-800，固定资产+800】，此为总资产不变的内部转换。
  * **交易12（计提折旧300）**：记【固定资产（累计折旧）减少300】，折旧费用记为【权益减少300】。

### 7. 业务场景：三大财务报表的数据汇总逻辑
* **知识点**：三大表的数据提取与时点/期间属性。
* **一句话总结**：资产负债表看“期末底行”，利润表看“期间变化右侧”，现金流量表看“时点+期间变化左侧”。
* **详细示例与证据**：
  * **资产负债表**：提取最后汇总计算得出的期末数（黑色代表资产，红色代表负债，绿色代表权益）。
  * **利润表**：纯粹的“期间概念”。提取工作表最右边（权益端）的期间变化量（正数是收入，负数是成本费用），不包含期初和期末的权益存量点。
  * **现金流量表**：“时点+期间”概念。除了提取最左边（现金端）经营、投资、筹资三大类的现金流变化，还必须加上期初现金，最终推导出期末现金余额。

# E. 问答环节

* **提问1：销售商品的成本总额（如1280），需要对应弄清楚是哪一笔明细销售的成本吗？**
  * **讲师解答**：想得太多了。做此类报表整体推演时，没必要知道单笔对应关系，记一个总数并保证资产负债表整体平衡即可。
  * **结论/影响**：培养“抓大放小”的报表全局观，避免陷入无关紧要的细节。
* **提问2：收到预付款时，现金进账了，为什么不能记收入？**
  * **讲师解答**：根据权责发生制，此时发货义务尚未完成，因此属于负债（预收账款）。只有在未来实际发货后，才能将预收账款转化为收入。
  * **结论/影响**：强化了“钱到不等于收入到，钱走不等于费用生”的权责发生制核心逻辑。

# G. 词汇表

| 术语名称 | 定义/解释 |
| :--- | :--- |
| 权责发生制 (Accrual Basis) | 以权利和义务的发生来决定收入和费用归属期的会计原则，而非以现金的实际收付为标准。 |
| 预收账款 (Unearned Revenue) | 企业按照合同规定向买方预收的款项，在未交付货物或提供服务前属于企业的负债。 |
| 应收账款 (Accounts Receivable) | 企业因销售商品、提供劳务等经营活动，应向购货单位或接受劳务单位收取的款项。 |
| 折旧摊销 (Depreciation & Amortization) | 将固定资产或无形资产的价值在其使用寿命内按一定方法进行系统分摊并计入期间费用的过程。 |
