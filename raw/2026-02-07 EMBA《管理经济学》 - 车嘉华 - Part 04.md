## D. 详细记录

### 1. 场景/问题：定价博弈决策分析（顾丽佳与开门红）
* **知识点**：利润贡献（Contribution Margin, R-VC）与会计利润的区别。
* **一句话结论**：在评估部门或产品线的短期定价决策时，应以最大化利润贡献（R-VC）为目标，而非减去摊销固定成本后的部门会计利润。
* **示例/证据**：
  * 教授引导学员计算定价为3元和4元时的情况。
  * `R（销售收入） - VC（可变成本） = 利润贡献`。
  * 在案例中，无论对手选3元还是4元，选择3元总能带来更大的 `R-VC`。
  * 教授给出一个多部门企业的类比（X, Y, Z部门），展示企业总成本 `TC` 中固定成本的摊销是与销量无关的沉没成本，若在决策中扣除这部分会导致决策失误。
* **❌ 常见误区**：认为总成本（TC = VC + FC）或单位摊销固定成本越低越好，并在比较定价方案时减去了摊销的固定成本。
* **✅ 纠正建议**：固定成本的分摊不应影响当前的边际定价决策，只看“销售额 - 可变成本”。

### 2. 场景/问题：需求曲线打造与弹性原则
* **知识点**：原则七（降低敏感度）与原则五（弹性区域选择）的结合应用。
* **一句话结论**：原则七是通过产品打造降低整体价格敏感度，原则五是无论曲线如何变化，最终的定价点仍应落在弹性大于1的区间。
* **示例/证据**：
  * **原则七**：星巴克门市店（蓝线：更平缓，敏感） vs 星巴克烘焙工坊（红线：更陡峭，不敏感）。通过体验升级（如烘焙工坊），企业改变了需求曲线的斜率和高度，降低了顾客的价格弹性。
  * **原则五**：对于任何直线需求曲线，中点的弹性为1，上半段（高价区）弹性 > 1，下半段（低价区）弹性 < 1。企业应避免在弹性 < 1的区域定价，因为提价能同时增加收入和降低成本。

### 3. 场景/问题：差异化定价——“量身定造”策略（第一度价格歧视）
* **知识点**：价值创造与价值分配、入门费/多阶定价。
* **一句话结论**：差异化定价的核心前提是防串货和了解客户信息，通过套餐、入门费等机制，企业能将创造出的“最大社会效益”全额转化为自身利润。
* **示例/证据**：
  * **SPA案例**：客户前四次的边际愿意支付为：1000, 800, 500, 200。边际成本为300。
  * 若采用单一定价800：做2次，利润为 `(800-300)*2 = 1000`。
  * 若采用阶梯定价/套餐（收2300做3次）：利润为 `(1000+800+500) - 300*3 = 1400`。
  * **模型解析**：红线以下是“客户愿意支付的总价值”，黄线以下是“可变成本”。两者之间的面积是“社会效益”。企业的最高利润上限即为该效益面积。为了获得最大利润，企业需先将销量推进到边际成本等于边际价值的点（三次，第四次200<300亏本不做），然后通过打包定价把所有的社会效益拿走。

## E. 问答记录

* **问题 1**：在顾丽佳案例中，为什么量越大的时候不应该看分摊的固定成本？
  * **解答要点**：固定成本作为整体已经发生，不论分摊给单件产品多少，总量不变。在边际决策（定价多卖一件）时，固定分摊只是账面游戏，真正影响现金流增加的是利润贡献（R-VC）。
  * **结论/影响**：改变了学员依据完全成本法（含摊销）进行战术定价的错误习惯。

* **问题 2**：如果客户觉得一次性收取高额套餐费（如10000元/20次）不合理怎么办？
  * **解答要点**：企业收取的总费用绝对不能超过“为客户创造的总价值（边际价值之和）”。如果创造的价值只有5700，收10000客户必然不会接受。
  * **结论/影响**：定价的上限受限于创造的社会效益。

## G. 词汇表

| 术语 | 定义/解释 |
| :--- | :--- |
| R (Revenue) | 销售收入。 |
| VC (Variable Cost) | 可变成本。 |
| 利润贡献 (R-VC) | 销售收入减去可变成本，衡量产品对填补固定成本和产生利润的边际贡献。 |
| 单一定价 | 对所有客户在所有场景下购买任意数量都采用同一单价的定价策略。 |
| 差异化定价 | 根据客户特征、购买数量或场景的不同，设定不同价格。分为量身定造、直接细分、间接细分。 |
| 量身定造 | 教授为本课程自创词汇（对应经济学中的“第一度价格歧视”或完全价格歧视），指针对每个客户的边际支付意愿收取最高价格。 |
