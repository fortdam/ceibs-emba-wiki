## B. 培训概述

  本次课程作为EMBA新生的“破冰”与导向指南，系统性梳理了中欧从1984年萌芽到如今“三洲五地”办学的历史脉络。重点阐述了在当前逆全球化和中国产业转型大背景下，EMBA课程体系向“全球视野、中国智慧、创新驱动、新商业文明引领者”的全新定位升级。同时，详尽规划了从开学模块、必修课、选修课（含实境课与海外模块）到毕业反思的完整学习生命周期。

## D. 详细记录

### 1. 认知中欧商学院历史与文化
* **核心知识点：** 中欧的发展历程与核心价值观。
* **一句话总结：** 中欧的发展史是中国经济发展与管理水平提升的缩影，其核心在于培养兼具社会责任、推动新商业文明的卓越领导者。
* **详细证据与解析：**
  * **历史沿革：** 
    * 前身可追溯至1984年改革开放初期的中国欧共体联合管理中心。
    * 1994年由上海市政府与欧盟委托的EFMD正式合作建校。
    * 2001年入世后加速发展，形成亚洲、欧洲、非洲的“三洲五地”办学模式。
  * **校训与使命：** 秉持“认真、创新、追求卓越”的理念。不仅追求商业成功，更致力于“培养兼具社会责任的领导者”，倡导“活着就是为改变世界”的企业家精神。
* **❌ 常见误区：** 认为商学院仅仅是传授商业技能和赚钱方法的场所。
* **✅ 纠正解析：** 中欧更看重社会责任的培养，致力于提升学员对社会进步的认知，点燃各行各业创新的“星星之火”。

### 2. 中欧EMBA项目的核心优势与课程体系设计
* **核心知识点：** EMBA项目的演进逻辑与“一体两翼”最新课程架构。
* **一句话总结：** 面对全球化2.0时代，中欧EMBA通过经典底座叠加前沿性、国际性、实践性，打造“一体两翼”全生命周期学习体验。
* **详细证据与解析：**
  * **项目演进阶段：** 
    * 1995-2001（萌芽起步）：首创中国内地EMBA，借鉴欧洲师资体系。
    * 2002-2008（系统构建）：建立独立师资，学员从外企/国企主导向民营企业家过渡。
    * 2009-2016（领导力转向）：增设“戈壁行动”等领导力反思课程，应对金融危机后的企业迷茫。
    * 2017至今（实战与出海深化）：开设海外模块与国内实境课，契合硬科技、智能制造等实体经济崛起。
  * **师资与教学特色（实境课）：** 
    * 师资具备极强的理论与实践结合能力（如担任名企独董）。
    * **实境课模式：** 教授开发案例 -> 带领全班深入企业现场 -> 与创始决策者交流 -> 教授提炼认知升华。每年选修课更新率达12%以上。
  * **全新课程体系（一体两翼）：**
    * **定位：** 全球视野、中国智慧、创新驱动、培育新商业文明引领者。
    * **“一体”（核心课程）：** 包含必修课（经典管理思想为底座，约1年，15门）+ 选修课（前沿性、国际性、实践性，70多门，分7大领域）。
    * **“两翼”：** 左翼为“第二课堂体系”（前沿分享、人文修养、实战演练）；右翼为“终身学习/后EMBA课程”（校友辅导、持续赋能）。
  * **学习时间轴：** 开学模块 -> 必修课（第1年） -> 半年后开始论文组队 -> 选修课与论文并行（约1.5年） -> 毕业反思模块。
* **❌ 常见误区：** 认为带着企业经营的具体疑难杂症来商学院，老师会直接给出标准答案或解决方案。
* **✅ 纠正解析：** 商学院不是咨询公司。中欧提供的是底层“思维框架”和开放性的思考能力，学员需通过学习提升全方位能力并自行解决问题。

### 3. 同行者画像与班级构成分析
* **核心知识点：** 2025秋季班学员数据透视。
* **一句话总结：** EMBA学员结构高度契合国家“十五五”产业导向，以民营实体经济、科技创新型企业的核心决策层为主流。
* **详细证据与解析：**
  * **企业属性：** 包含11家超大规模企业，69家世界/中国500强，120家上市公司，50家国家级专精特新“小巨人”。
  * **行业分布：** 核心聚焦于智能制造、信息技术（IT）、医药健康三大板块。
  * **所有制与人员结构：** 民营企业占绝对主导（81.5%），外资占9%，国企占6.6%；女性高管比例达到35%。

### 4. 学习建议与行为规范（有所为与有所不为）
* **核心知识点：** PTSCD全方位中欧体验与红线管理。
* **一句话总结：** 在中欧的两年需秉持空杯心态互学互鉴，恪守学术与商业道德底线，实现身心与心智的全面蜕变。
* **详细证据与解析：**
  * **有所为（建议项）：**
    * **度过阵痛期：** 头半年必修课理论性强、挑战大，需全力投入，切勿用“经验论”抵触理论学习。
    * **空杯心态与分享文化：** 放下既往光环，通过与教授吃“盲盒饭”、与同学私董会实现“教学相长”。
    * **融入班级建设：** 积极配合班主任（中欧核心运营资产），选优班委，塑造可延续十年的优良班级文化。
    * **践行公益与健康：** 至少参与一项公益活动（承担社会责任）；迈开腿参与体育运动，培养健康生活方式。
  * **有所不为（红线项）：**
    * 严禁迟到早退、无故旷课（跨班补课体验极差）。
    * 坚决杜绝考试作弊与论文抄袭，勿将成绩看得比真实反思更重。
    * 禁止在高铁等公共场合大声喧哗商业机密或炫耀，维护中欧品牌声誉；严守法律底线。

## E. 问答环节
*因开学第一课为统一宣讲，现场未记录具体互动问答，但讲师对潜在的共性疑问进行了预答：*
* **问题（预设）：如果我对个别教授的授课风格不适应怎么办？**
  * **解答：** 中欧有极其卓越的“中欧三许”（徐立波、许满、许小年等）作为标杆，但这不代表所有教授都是同样风格。部分年轻教授或侧重于前沿理论深度（如AI领域），部分更侧重于研究。
  * **结论建议：** 建议学员包容多样性的授课风格，重点汲取教授在“思想深度”和“学术框架”上的营养。如果整个EMBA有80%的课程让大家觉得获益匪浅，即为极大成功，不必苛求100%完美。

## F. 待办事项与资源
* **推荐行动：**
  * 做好心理建设，全力以赴应对前半年专业性极强、挑战度高的“必修课”阶段。
  * 入学半年后（大约时段），需完成小组组建，开启毕业课题的调研与撰写（耗时约1.5年）。
  * 至少规划并参与一项社会公益活动，践行企业家社会责任。
* **相关资源：**
  * 可前往中欧图书馆借阅《中欧校史》等文献，深化对学院底蕴的了解。
  * 积极利用“实境课”下企业调研的资源，以及由25位行业领袖组成的“首届中欧EMBA顾问委员会”提供的智慧参考。

## G. 专业术语表

| 术语名称 | 详细释义 |
| :--- | :--- |
| **EFMD** | 欧洲管理发展基金会（European Foundation for Management Development），中欧建校初期的重要欧方合作机构。 |
| **三洲五地** | 指中欧在亚洲、欧洲、非洲设立的五个校区/办学点（包括上海、北京、深圳、瑞士苏黎世、加纳阿克拉）。 |
| **实境课** | 中欧特色教学模式。由教授开发案例，带领全班深入企业现场与高管直接交流，最后由教授提炼认知的沉浸式学习法。 |
| **一体两翼** | 中欧EMBA最新课程体系。一体指核心课程（必修理论底座+选修前沿实践）；两翼指第二课堂（前沿/人文分享）与后EMBA终身学习体系。 |
| **PTSCD模式** | 指代全方位的中欧体验（Total of Experience），不仅涵盖学术知识，还包括健康生活、社会责任、人脉圈层等多维度蜕变。 |
