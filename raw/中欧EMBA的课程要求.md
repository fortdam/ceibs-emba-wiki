# 中欧EMBA课程要求说明会

### **会议信息**
*   **日期:** 2025年10月27日
*   **时间:** 19:30 - 20:03
*   **参会者:** 2025级秋季三个上海班A组的同学, 班主任, 徐一帆老师 (5班班主任), 袁丽娟老师 (6班班主任)

### **简要总结**
本次会议是针对中欧EMBA 2025级秋季上海班同学的课程要求说明会。班主任详细介绍了EMBA项目为期两年的课程结构、毕业必须满足的四项核心要求、严格的学术纪律与考勤制度，以及丰富多彩的校园活动。会议强调了课程学习的全过程管理（课前、课中、课后），并对成绩评定、学分获取、调课政策、学术诚信等关键规定做出了详细解释，旨在帮助新生快速适应学习生活，顺利完成学业。

### **议题与讨论**
#### **议题一：EMBA课程之旅概览**
*   **讨论与分析:**
    *   **课程结构:** 整个学习旅程为期两年，共需完成60个学分。
        *   **开学模块:** 当前所处的阶段，日程丰富。
        *   **必修阶段 (约1年):** 系统学习十几门商业管理核心课程，以战略思维为引领。
        *   **选修阶段 (约10个月):** 提供超过一百门次的选修课，涵盖七大管理实践领域，包括国内实践课和海外选修课。
        *   **ESG必修项目:** 为期一天，分阶段进行，通常安排在某一门必修课前。
        *   **小组课题项目:** 以小组形式完成，旨在运用所学知识解决企业真实管理问题。
        *   **毕业就读模块:** 完成两年学习之旅的最终环节。
    *   **学习流程:** 强调课前预习、课中参与、课后复习的有效学习流程。
        *   **课前:** 通常提前一个月发放阅读材料。建议利用碎片化时间阅读，并参考案例法学习视频。财务课程建议先观看在线预习课件。
        *   **课中:** 正式上课期间，要求专心听讲，积极参与，禁止录音录像。
        *   **课后:** 课程结束会有考试或课后作业，并邀请学生填写课程反馈。

#### **议题二：毕业要求与学术规定**
*   **讨论与分析:**
    *   **毕业四要素:**
        1.  **学分:** 修满60个学分。
        2.  **绩点:** 所有课程的平均绩点不低于2.8。
        3.  **成绩:** C或F的课程不能超过两门。
        4.  **学术查看:** 符合学术查看绩点的规定。
    *   **学分获取条件:**
        *   单门课程成绩在C+及以上。
        *   考勤达标。
        *   遵守学术纪律。
    *   **成绩评定:**
        *   **组成:** 成绩由考试/课后作业、课前测验、出勤、课堂参与、案例分享等按比例加总构成。
        *   **绩点系统:** 字母分对应不同绩点（如A=4.0, B-=2.7, C+=2.3）。若单科成绩低于B-，需在后续课程中努力提升平均绩点至2.8以上。
        *   **正态分布:** 每门课的成绩以班级为单位，参照一定比例呈正态分布。
        *   **不及格处理:**
            *   **能力问题 (C或F):** 若因考试/作业未发挥好，有一次重考或重做机会，通过后成绩可改为C+。
            *   **态度问题 (C或F):** 若因考勤不达标、作业迟交20天以上、违纪违规等，必须付费重修。重修不得超过两门，每门课仅限重修一次。
    *   **学术查看绩点:**
        *   违纪或成绩不佳会产生学术查看绩点（如C+产生0.5点）。
        *   累计到2点会收到警告，2.5点将进行退学评估，累计到4点直接退学。
*   **决定:**
    *   学生必须严格遵守毕业要求和学术规定，以确保能顺利毕业。

#### **议题三：考勤与调课政策**
*   **讨论与分析:**
    *   **考勤制度:** 制度非常严格。
        *   **迟到:** 上课时间一到即开始计算，如9:00上课，9:02到教室记为缺勤半小时。
        *   **缺勤上限:**
            *   两天的课程：缺勤不得超过4小时。
            *   三天的课程：缺勤不得超过6小时。
            *   四天的课程：缺勤不得超过8小时。
        *   **累计缺勤:**
            *   两年内所有课程缺勤时间会累积。
            *   累计24小时：预警。
            *   累计32小时：需付费加修一门1.5学分的课程。
            *   累计64小时：将被退学。
        *   **强制休学:** 持续缺席六门课程或连续六个月不到校，将被强制休学。
    *   **调课政策:**
        *   **定义:** 指整个模块无法参加，需换时间到其他班级补课。
        *   **申请:** 需提前至少五天在调课系统中在线申请。
        *   **影响:** 调课会影响正常学习节奏，且补课席位不确定，可能影响按期毕业。
        *   **调课点数:** 入学时每人有200初始点数，每次调课会扣除相应点数。点数将用于第二年选修课的抽签，点数少可能导致选不到心仪的课程。
*   **决定:**
    *   强烈建议学生按时上课，避免频繁请假和调课。如有不得已情况需要调课，必须提前与班主任沟通。

#### **议题四：学术诚信（红线规定）**
*   **讨论与分析:**
    *   **考试作弊与作业抄袭:** 均属不可触犯的“红线”，一经发现将严肃处理。
    *   **抄袭定义:**
        *   禁止使用AI生成内容或大量借鉴网络资料。
        *   同一份自己写的作业不可用于两门不同的课程。
        *   学校有反抄袭系统对每份作业进行查重。
    *   **后果:**
        *   成绩记为F，全班通报。
        *   影响竞选班委和评选优秀毕业生的资格。
    *   **作业提交:**
        *   需按统一格式和规定时间提交。
        *   迟交10天内，成绩降一档。
        *   迟交20天内，成绩降两档。
        *   迟交20天以上，作业不予接收，该门课程不及格，需付费重修。
*   **决定:**
    *   所有学生必须独立完成作业，杜绝任何形式的抄袭和作弊行为。

#### **议题五：多彩的校园生活与活动**
*   **讨论与分析:**
    *   **学校层面:** 提供大师课堂、高端论坛、艺术人文赏析等大型活动，并支持学生自主组织的活动（如红枫路体育学院、新春音乐会）。
    *   **项目办公室层面:** 组织“一家人脱口秀”、人文艺术讲座、合一提堂（关注前沿热点）、行知课堂（探讨管理实战）以及鸿鹄论坛等。
    *   **班级层面:** 鼓励各班组织班级论坛、企业参访、短途游学等，并强调同学间的相互学习（50%向教授学，50%向同学学）。
    *   **公益责任:** 每个EMBA班级都会开展公益项目，传递社会责任感。学校每年评选“善为奖”以表彰杰出贡献的班级与个人。
*   **决定:**
    *   鼓励学生积极融入中欧的各类学习和实践活动中，充分利用平台资源，丰富学习体验。
