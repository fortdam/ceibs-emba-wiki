# D. 详细记录

### 1. 场景/核心问题：如何评估动机的“质量”而非单纯“强弱”？
* **知识点：** 自我决定论（SDT 2.0）及三大基本心理需求（自主性、能力感、归属感）。
* **一句话总结：** 高质量的动机不仅仅依赖于“是否有动力”，而是取决于个体的三大基本心理需求是否得到满足。
* **案例/论据：**
  * **斯坦福大学实验：** 对两组画画的儿童，一组给予画画奖励（棒棒糖），另一组无奖励。结果表明，外部奖励反而削弱了儿童对画画本身的内在动力，导致其只关注奖励而非画画本身。
* **常见误区：**
  * ❌ 误区：给员工越多的金钱激励，员工的动力就一定越强且持久。
  * ✅ 纠正：单纯的外在物质刺激（单纯型外在动机）可能会伤害内在动机；一旦激励取消，行为也会随之停止。

### 2. 场景/核心问题：外在动机如何向上层级转化（内化）？
* **知识点：** 外在动机的四个层级（单纯型、内设型、认同型、整合型）。
* **一句话总结：** 管理者的核心目标是帮助员工将单纯的“为奖金工作（单纯型）”和“怕被批评（内设型）”的低质量外在动机，内化为“认可工作价值（整合型）”的高质量外在动机。
* **案例/论据：**
  * **合规管理案例：** 员工为了奖金做合规（单纯型）；因为老板在群里点名批评而连夜补材料（内设型）；认为合规有长期价值但自身未必完全接受（认同型）；将合规视为职业操守底线（整合型）。
* **常见误区：**
  * ❌ 误区：内在动机等同于最高层级的外在动机。
  * ✅ 纠正：内在动机是纯粹的个人兴趣和满足感（如热爱做航模）；而整合型的外在动机仍需要外部肯定，但关注点是该工作的“意义感”和价值认同。

### 3. 场景/核心问题：如何挖掘和落地企业的“核心价值观（Why）”？
* **知识点：** 黄金圈法则（Why-How-What）与反向推导法。
* **一句话总结：** 明确企业的核心信念（Why），可以自然推导出战略路径（How）并有效筛选具有共同价值观的员工，降低管理成本。
* **案例/论据：**
  * **拼多多案例：** 核心价值观是“关注中国最广大的老百姓（Why）”，由此推导出“深耕下沉市场（How）”，最终成为了“高速成长的电商龙头（What）”。
  * **vivo案例：** 遇到翻盖机向智能机转型的巨额沉没成本，坚持“本分”的价值观，选择自己承担损失而不违约转嫁给供应商。

### 4. 场景/核心问题：信任缺失对组织带来的管理隐患是什么？
* **知识点：** 信任成本效应与信任红利（信任治理理论）。
* **一句话总结：** 组织内缺乏信任会导致防御性管理动作增多（如过度抄送邮件、反复审计），从而增加间接成本并扼杀创新。
* **案例/论据：**
  * 某国立大学员工发邮件要抄送三十多人，以规避责任。
  * 缺乏信任的组织只能用流程、合同等高成本工具来替代信任，造成决策缓慢、客户流失。

# E. 问答环节

**问：在派遣高管去马来西亚开辟新市场时，如何同时满足公司业绩需求与高管的心理预期？（角色扮演环节总结）**
* **回答要点：**
  * **自主性：** 给予在海外市场充分的人事、预算及一定的战略决策自主权（如一定额度内的审批权放宽）。
  * **归属感：** 允许高管保留集团职务，定期参加集团管理层会议，避免产生被边缘化或“流放”的感受。
  * **能力感：** 明确表示指派是基于对其能力的最高信任，并设立可达成的阶段性目标以建立正向反馈。
* **结论/影响：** 如果无法将高管的动机从单纯的物质追求（单纯型外在动机）拉升至对战略意义的认同（整合型外在动机），仅靠砸钱（如薪酬翻三倍）无法买到忠诚与高效；高管一旦遇到更好待遇便会跳槽，或者仅将调动视为一种条件交换。

# G. 术语表

| 术语 | 定义/解释 |
| :--- | :--- |
| **马斯洛需求层次** | 经典的动机理论，认为人类需求按层级排列，底层生理和安全满足后，向归属、尊重及自我实现递进。 |
| **ERG理论** | 简化版需求理论，将需求归纳为生存(Existence)、相互关系(Relatedness)和成长(Growth)。 |
| **预期理论(Expectancy Theory)** | 动机强度取决于：期望（E）× 媒介/工具性（I，付出产出比）× 效价（V，结果的重要性）。 |
| **SDT自我决定论2.0** | 区分动机质量的理论，强调行为受自主性、能力感和归属感三大心理需求驱动。 |
| **外在动机四层级** | 自低向高分别为：单纯型（为奖罚）、内设型（为避免羞愧或被比较）、认同型（理解价值）、整合型（视作个人底线与意义）。 |
| **黄金圈法则(Why-How-What)** | 从内向外的思考模式；伟大组织往往先搞清核心价值（Why），再落实方法（How）和具体动作（What）。 |
